Security Mountain is an open-source community driven project aiming to help individuals and businesses increase their online security posture, step-by-step.

To run the project locally:
<code>
npm install yarn
</code>
<code>
yarn install gatsby
</code>
<code>
yarn gatsby develop
</code>
